# 功能目标

对输入的机械工程图纸进行自动化识别、几何语义解析，将图纸中所有标注、技术要求、图纸属性转换为**带唯一编号的结构化技术要求项目清单**，结果需完整覆盖图纸核心信息，同时输出标注坐标信息，支撑后续我方自主完成序号修改、检验计划生成、图数双向定位等业务功能。

# 输入兼容要求

1.支持格式：兼容矢量 PDF、300DPI 及以上清晰度的扫描版 PDF，以及 DWG/DXF 等 CAD 格式导出的图纸文件；

2.支持模式：支持单张图纸上传识别、批量图纸上传识别（单次批量≥50 张），适配多页长图纸分页识别。

# 输出字段要求

输出结果分为「图纸全局属性」和「单条技术要求项目」两部分，所有必填字段不得遗漏：

## 图纸全局属性（每张图纸输出 1 组）

| **字段名** | **输出要求** |
| --- | --- |
| 产品名称 | 准确提取标题栏中产品 / 零件名称，完整识别中英文、数字、特殊符号 |
| 图纸号 | 准确提取标题栏中图号 / 零件号，支持带前缀、后缀、版本标记的复杂图号识别 |
| 版本号 | 提取标题栏中图纸版本 / 修订号，兼容 A/0/1/Rev.1 等常见版本格式 |
| 页码信息 | 输出当前页页码、图纸总页数，多页图纸分页独立识别 |
| 图幅信息 | 识别图纸幅面规格（A0/A1/A2/A3/A4）、图纸方向（横向 / 纵向） |
| 全局技术要求 | 完整提取图纸中全局通用要求，包括未注公差标准、全局粗糙度、热处理要求、通用倒角规则等全图生效的技术条款 |

## 单条技术要求项目（每个标注 / 独立技术条款输出 1 条，自带唯一编号）

| **字段名** | **输出要求** |
| --- | --- |
| 项目初始编号 | 为每个识别出的标注 / 技术条款生成全局唯一数字编号，编号与图纸标注一一对应，无重复、无遗漏 |
| 图幅区域 | 识别该标注所在的标准图幅分区（按 GB/ISO 行列分区规则，如 A1、B3、C2） |
| 标注类型 | 标记标注分类：尺寸标注（长度 / 半径 / 直径 / 角度 / 螺纹 / 孔位等）、尺寸公差标注、形位公差标注、粗糙度标注、焊接符号标注、技术要求文本、其他特殊标注 |
| 标注内容 | 完整提取标注的数值 / 文本内容：- 尺寸类：尺寸数值、单位、螺纹规格（如 M6、G1/2）- 公差类：公差类型、公差值、基准信息（如位置度 Φ0.1 @A）- 文本类：完整提取技术要求文本，保留换行、条目逻辑 |
| 数量 | 针对成组标注（如4-Φ10、6 处均布孔、阵列特征等），单独识别并输出对应特征的数量 |
| 公差详情 | 拆分输出独立字段：1. 标注自带公差的上偏差、下偏差；2. 未注公差自动匹配结果：根据图纸指定标准（ISO/GB/ASME 等），输出对应尺寸的未注公差值、引用标准号 |
| 唯一标识 ID | 为每个项目生成全局唯一字符串 ID，作为后续序号修改、数据关联、量检具绑定的唯一主键 |
| 坐标信息 | 输出该标注在图纸页面坐标系下的完整坐标：包含标注锚点坐标、文本框四角坐标、引线起止点坐标；需明确坐标系规则（原点为页面左下角、单位统一为 mm/pt、自动适配页面旋转），坐标与实际位置偏差≤2mm，确保我方后续可基于坐标完成序号气泡添加、位置修改等操作功能 |

## 功能要求

* + 1. 标注与清单配套能力

坐标标准化输出

1. 所有标注统一输出一套完整坐标集：标注锚点、尺寸线起点 / 终点、公差文本包围盒、气泡引线坐标；统一坐标系原点、单位（mm）、旋转矫正参数，坐标偏移误差≤1pt，支持我方基于坐标自主修改序号、绘制气泡标注，第三方无需实现序号编辑功能。
2. 支持清单条目与图纸双向映射绑定，通过唯一 ID 关联坐标，支撑双击跳转定位功能数据底座。

多类型图纸兼容增强 兼容矢量 PDF、扫描件 PDF（300DPI 及以上）、DWG/DXF；支持老旧模糊图纸、轻微手写标注、多语言混合图纸（中英德）识别；图纸自动倾斜矫正、污渍遮挡区域标记。

1. 数据导出结构化规范 识别结果统一输出标准 JSON，完整包含图纸元数据、清单条目、坐标、置信度、公差匹配结果；兼容我方插件直接解析，预留扩展字段用于后续对接 PLM/QMS。
   * 1. （二）接口集成配套功能（B/S 架构强制）
2. 提供 RESTful HTTP 标准化 API，接口清单包含：图纸上传接口、任务查询接口、结果获取接口、标准库更新接口、批量任务管理接口；
3. 接口支持鉴权、请求加密、传输加密，所有操作生成可追溯日志；
4. 支持插件轻量化调用，无需客户端本地安装重型软件，适配平台嵌入式集成模式。
   * 1. （三）异常与复核配套功能
5. 每条识别结果输出 0~100 分置信度，置信度＜80 分自动标记「人工复核项」，单独汇总输出复核清单；
6. 识别失败、图纸损坏、加密图纸、模糊无法解析图纸返回标准化错误码 + 失败原因；
7. 支持局部遮挡、残缺图纸完成可识别区域解析，仅标记失效条目，不中断整张图纸识别流程。

**二、硬性性能指标要求（可量化验收）**

**1. 识别速度指标**

1）单张图纸≤100 个尺寸标注：完整解析（含坐标提取、未注公差匹配、结构化清单生成）耗时≤3min； 2）单张图纸 100~300 个尺寸标注：解析耗时≤6min； 3）批量任务：并行处理，平均单张处理耗时不超过上限，排队延迟不超过 10min。

**2. 并发承载指标**

1）服务端支持**30 个并发用户**同时发起图纸识别请求； 2）高并发场景下无任务丢失、无数据错乱、接口响应超时率＜0.5%； 3）支持水平扩容，扩容后并发能力线性提升。

**3. 稳定性指标**

1）图纸加载、解析失败率≤1%； 2）7×24 小时连续运行，无内存泄漏、进程卡死； 3）批量任务中断后支持断点续跑，已识别数据不丢失。

**4. 图纸文件兼容性能**

1）支持单页图纸最大尺寸 A0 大图、多页图纸单次上传 200 页以内； 2）单文件上限 500MB 扫描 PDF、200MB 矢量 PDF/DWG，超大文件分片上传解析。

**三、识别精度 & 准确率硬性指标（验收核心）**

整体图纸识别综合准确率≥90%（以人工全检标注清单为基准），分层细化约束：

1. 图纸全局关键字段（产品名称、图纸号、版本、页码、图幅区域）识别准确率≥95%，不允许关键信息错漏；
2. 标准打印体尺寸、数值、公差、螺纹规格识别准确率≥92%；
3. 形位公差符号、粗糙度、焊接符号识别准确率≥91%；
4. 清晰手写补充标注、老旧褪色图纸识别准确率≥85%，低置信度条目强制标记复核；
5. 坐标点位精度：标注文本 / 引线坐标与图纸实际位置偏差≤1pt，满足我方序号绘制、双向定位使用；
6. 未注公差自动匹配正确率≥98%，标准匹配无错配、漏配。

**四、安全、日志、可运维补充指标**

1. 全链路日志：图纸上传、识别、接口调用、数据查询全流程生成操作日志，包含账号、时间、图纸编号、操作内容，日志留存≥180 天；
2. 敏感图纸数据传输、存储加密，满足审计与数据安全要求；
3. 提供备份恢复能力，识别任务缓存、标准库配置支持一键备份 / 还原；
4. 支持监控指标输出：并发量、任务耗时、识别失败率、低置信条目占比，便于运维监控。

**五、部署集成配套指标**

1. B/S 架构场景：提供全套网络 API 接口文档、调用示例，支持我方平台插件无侵入集成；
2. 本地部署场景：提供 SDK / 本地调用接口，支持分布式、多租户集中部署模式；
3. 标准库可离线更新，无需联网即可导入新版 ISO/GB 公差标准；
4. 识别输出数据预留扩展字段，可无缝对接 PLM 图纸拉取、QMS 检验方案同步业务链路。

**六、验收判定标准**

同时满足以下全部条件视为该模块达标：

1. 随机抽取 100 张覆盖各类图纸（扫描件 / CAD、新旧图纸、多标准、多图幅），综合识别准确率≥90%；
2. 100 尺寸以内图纸解析耗时≤3min，30 并发稳定运行无报错；
3. 图纸加载失败率≤1%，坐标偏移≤1pt；
4. 完整输出全部要求字段、坐标、置信度、公差匹配数据；
5. API/SDK 可稳定对接我方平台插件，批量、版本对比、日志加密功能全部可用。
